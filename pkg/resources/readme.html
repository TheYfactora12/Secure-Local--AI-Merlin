After Installation — Next Steps

1. Add your API keys (optional)

cd ~/home-ai-elite
nano .env

Cloud API keys are optional. The stack works fully offline with local models.

2. Run first-boot initialization

bash ~/home-ai-elite/scripts/bootstrap.sh

This sets up Qdrant collections and imports n8n workflows automatically.

3. Open your services

  • Open WebUI — http://localhost:3001
  • n8n Automation — http://localhost:5678
  • Perplexica Search — http://localhost:3000
  • Qdrant Dashboard — http://localhost:6333/dashboard
  • SearXNG — http://localhost:8080

Useful commands

# Check all services
bash ~/home-ai-elite/scripts/status.sh

# Safe upgrade (with auto-rollback)
bash ~/home-ai-elite/scripts/upgrade.sh

# Stop everything
bash ~/home-ai-elite/scripts/stop.sh

# Uninstall
bash ~/home-ai-elite/pkg/scripts/uninstall.sh

Troubleshooting

  • Install log: /tmp/homeai-install.log
  • Service logs: docker compose logs [service-name]

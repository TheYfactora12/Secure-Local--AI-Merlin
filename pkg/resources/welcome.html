🧠 home-ai-elite

Your private, self-hosted AI stack — a local Perplexity + Codex + memory system that runs entirely on your Mac. No cloud required.

What gets installed

  • Open WebUI — Chat UI for all your local models
  • Ollama — Local LLM engine (Llama, Mistral, Gemma, DeepSeek)
  • Perplexica — Private AI-powered web search
  • n8n — AI automation & workflow engine
  • Qdrant — Vector memory store for RAG
  • SearXNG — Private meta-search (Google/Bing/DDG, no tracking)
  • LiteLLM — Model router (local-first, cloud fallback)

Requirements

  • macOS 13 Ventura or later
  • 8GB RAM minimum (16GB+ recommended)
  • 20GB free disk space
  • Docker Desktop (will be checked before install)

What this installer does

  • Copies stack files to ~/home-ai-elite
  • Runs install.sh in the background
  • Installs auto-start agents (services start on every login)
  • Places a Next Steps file on your Desktop

# Merlin AI — Go-To-Market Strategy

Last updated: 2026-05-10

## One Sentence

Download one file, run it, and in 30 minutes you have a fully private AI running
on your Mac that you own forever. If you ever want it gone, one click removes
everything.

## Beachhead Buyer

The first buyer is not an enterprise committee.

The first buyer is a privacy-conscious Mac owner who has heard enough about
cloud AI data use, wants ChatGPT-style help at home, and does not want to spend
hours learning Docker, models, and ports.

They will pay when the product proves:

- install is easy,
- the first screen makes sense,
- cloud is off by default,
- failures explain themselves,
- uninstall is clean.

## What We Sell First

Sell v1.0 as:

> Your private AI. No cloud account. No subscription. Merlin inside.

Do not sell v1.0 as:

- an enterprise platform,
- a compliance solution,
- a home automation system,
- an autonomous agent system,
- a developer toolkit,
- a public release.

## v1.0 Demo

1. Download/run Merlin AI.
2. Wizard HQ opens.
3. User sees "Your private AI is ready."
4. User asks Merlin a local question.
5. User sees cloud is off by default.
6. User sees recovery guidance if anything is degraded.
7. User sees how to uninstall or purge.

If this demo is not clean, pricing and expansion do not matter yet.

## Pricing Hypothesis

Start simple:

- Merlin AI Core: one-time consumer price after signed installer and clean
  evidence exist.
- Optional support/update plan later.

Do not publish pricing until 10 real nontechnical install tests pass.

## Future Markets

Regulated professionals, small practices, schools, MSPs, smart-home users,
Linux users, and professional evidence mode are future. They are valid, but they
wait in [`FUTURE_IDEAS.md`](FUTURE_IDEAS.md) until consumer v1.0 is proven.

## Current Go-To-Market Work

1. Make README and installer language nontechnical.
2. Build the first-run onboarding surface.
3. Produce clean install/uninstall evidence.
4. Record a short demo only after the above works.
5. Then test with 10 real users.

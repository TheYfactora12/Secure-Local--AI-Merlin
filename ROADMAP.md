# Merlin AI Roadmap

The active roadmap lives in
[`docs/MERLIN_IMPLEMENTATION_ROADMAP.md`](docs/MERLIN_IMPLEMENTATION_ROADMAP.md).

v1.0 has only five focus areas:

1. Install that just works.
2. Privacy that is architecturally enforced.
3. Onboarding that removes confusion.
4. Uninstall that builds trust.
5. Open source credibility.

Everything else is future unless it directly strengthens one of those five
outcomes.
